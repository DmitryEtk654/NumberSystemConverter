# Конвертер систем счисления
- Приложение для перевода чисел между двоичной, восьмеричной, десятичной и шестнадцатеричной системами счисления.

## Возможности
- Конвертация чисел между системами 2, 8, 10, 16
- Валидация вводимых символов
- Сохранение истории конвертаций в SQLite
- Экспорт истории в CSV и JSON
- Загрузка и отображение изображений (Pillow)
- Автоопределение системы по префиксам (0b, 0o, 0x)
- Копирование результата в буфер обмена
- Современный интерфейс с QSS-стилями
- Светлая и тёмная темы

## Горячие клавиши
- Ctrl+Enter	# Конвертировать число
- Ctrl+Shift+C	# Очистить поля ввода и результат
- Ctrl+C	# Копировать результат в буфер обмена

## Установка и запуск
1. Создать виртуальное окружение
- python -m venv venv

2. Активировать виртуальное окружение
- Windows: venv\Scripts\activate
- macOS / Linux: source venv/bin/activate

3. Установить зависимости
- pip install -r requirements.txt

4. Запустить приложение
- python main.py
